# 项目页面原型

KinEcho 当前目标是一个小程序端加一个后台监控端。

## 小程序端

小程序包含三类角色：

- 老人端：AI 语音陪伴、心情记录、提醒、回忆播放、求助、个人设置
- 家属端：老人状态概览、互动历史、留言、媒体上传、提醒和预警处理
- 服务人员端：工作台、个案、任务、随访、个人信息

老人端 AI 陪伴的核心链路：

```text
老人说话
-> 小程序录音
-> Java 后端语音识别
-> AI 生成回复
-> Java 后端合成语音
-> 小程序播放语音回复
```

## 后台监控端

后台监控端保留在 `src/admin`，用于查看平台运营、老人数据、预警、服务协同和统计分析。

## 不再保留

旧的 H5 老人端、H5 家属端、外部渲染页面、Python 语音服务和 MCP 服务已经从当前主结构中移除。
