# 小程序关键路径冒烟测试清单

适用范围：微信开发者工具打开 `miniapp`，后端 Java 服务使用 `http://127.0.0.1:8000/api`，默认家庭档案为 `family_001`。

## 测试前准备

- Java 后端已启动，访问 `/api/health` 返回 `status=ok`。
- 小程序已重新构建：`cd miniapp && npm run build:weapp`。
- 已执行冒烟预检：`npm run smoke:miniapp`；如需强制校验本地后端健康状态，执行 `cd miniapp && npm run smoke:checklist -- --require-api`。如需验证用户真实新增、编辑、软删除链路，使用开发库执行 `cd miniapp && npm run smoke:checklist -- --require-api --live-user-crud`。
- 微信开发者工具打开 `miniapp` 目录，并清理一次小程序缓存后重新进入。
- 默认演示数据已初始化，至少包含 1 位老人、1 位家属、若干提醒和咨询师。

## 登录与角色

| 编号 | 场景 | 步骤 | 预期 |
| --- | --- | --- | --- |
| L-01 | 角色选择 | 打开小程序，依次点击老人端、家属端、服务端角色卡片 | 跳转到对应登录页，页面标题和主题色与角色一致 |
| L-02 | 老人端登录 | 选择老人端，输入任意用户名和密码，点击登录 | 保存老人端会话并进入老人首页 |
| L-03 | 家属端登录 | 返回角色选择，选择家属端，输入家属姓名或手机号和密码后登录 | 调用 `/api/auth/login`，保存家属会话并进入家属看板 |
| L-04 | 服务端登录 | 返回角色选择，选择服务端，输入服务账号和密码后登录 | 调用 `/api/auth/login`，进入服务人员工作台，底部导航可见 |
| L-05 | 管理端登录 | 打开后台管理端，输入后台账号密码后登录 | 调用 `/api/auth/login`，登录成功后显示后台首页与当前管理员信息 |

## 老人端

| 编号 | 场景 | 步骤 | 预期 |
| --- | --- | --- | --- |
| E-01 | 首页数据同步 | 进入老人首页，下拉刷新 | 今日提醒、家人留言、今日回忆从 Java 后端同步；失败时展示错误 Toast |
| E-02 | 今日提醒完成 | 首页点击一条未完成提醒的“完成”按钮 | Toast 显示已完成，提醒状态刷新；家属端护理计划同步变化 |
| E-03 | 情绪记录入口 | 首页右上角心情按钮进入记录页，选择情绪后保存；点击“查看详细趋势” | 写入情绪记录，家属端情绪页和周报可看到新增数据；详细趋势进入独立历史记录页，不在当前页原地展开 |
| E-04 | AI 陪伴 | 进入陪伴页，发送文字消息；如有录音权限，再测试语音输入 | 消息进入对话流；后端可返回 AI 文本/音频时正常播放，否则给出明确错误 |
| E-05 | 回忆播放 | 进入回忆页，点击“播”、上一条、下一条、喜欢、语音讲述 | 图片打开预览，视频调用播放器，播放记录和喜欢反馈写入后端 |
| E-06 | 媒体播放页 | 从回忆或媒体页进入详情，切换内容并提交喜欢/不喜欢 | 播放记录、反馈状态和历史页同步更新 |
| E-07 | 求助联系 | 进入联系家人页，点击通知家人、拨打电话、紧急求助 | 普通联系生成中等级通知；紧急求助生成高等级通知；拨号调起系统能力 |
| E-08 | 心理咨询 | 进入心理咨询页，切换咨询方式并预约咨询师 | 新建咨询记录，家属端咨询页可看到同一条预约 |
| E-09 | 基本信息维护 | 在我的页点击基本信息，修改姓名/电话并保存后返回我的页 | 保存调用 `/api/users/{userId}`，我的页头像、姓名和电话同步刷新 |
| E-10 | 绑定家属管理 | 在我的页点击绑定家属，查看并复制绑定码；让家属端使用绑定码完成关联，再回到当前页删除一位已绑定家属 | 老人端调用 `/api/users/{userId}/binding-code` 获取绑定码；家属端调用 `/api/users/bind-by-code` 完成关联；删除仍调用 `/api/users/{userId}`，求助中心联系人同步变化 |
| E-11 | 我的页设置 | 在我的页依次点击字体、高对比、播报、语言、AI 陪伴、情绪记录、回忆播放、同步状态 | 每个入口都有选择器、跳转或刷新反馈，不出现无响应按钮 |
| E-12 | 偏好全局生效 | 在我的页切换字体和高对比后，切到首页、陪伴、记录、提醒、回忆、媒体历史 | 老人端所有页面应用同一字体/高对比偏好，重新进入小程序后仍保留 |

## 家属端

| 编号 | 场景 | 步骤 | 预期 |
| --- | --- | --- | --- |
| F-01 | 看板概览 | 进入家属看板，下拉刷新 | 情绪、提醒完成、最近播放和预警数据从后端同步 |
| F-01A | 绑定老人 | 进入我的页，点击“绑定老人”，输入老人端提供的绑定码、家属姓名和电话 | 绑定成功后家庭会话切换到目标家庭档案，我的页显示已绑定老人 |
| F-02 | 留言安排 | 新建一条留言，切换到老人首页 | 老人首页可见家人留言，老人点击“我已收到”后家属端状态更新 |
| F-03 | 护理计划 | 新建计划、标记完成、跳过、删除 | 老人提醒页与家属护理页状态一致 |
| F-04 | 预警处理 | 进入通知页，查看未读、标记已读、处理预警 | 预警状态变化写入后端，服务端工单状态同步 |
| F-05 | 媒体上传 | 上传图片或视频，进入详情编辑标签、时段、情绪策略并保存 | 老人端回忆推荐可看到该内容；策略保存后再次进入仍保留 |
| F-06 | 媒体删除 | 删除一条媒体内容 | 家属端列表移除，老人端推荐不再出现该内容 |
| F-07 | 咨询协同 | 新建心理咨询预约，推进咨询状态 | 老人端和服务端咨询/随访记录保持一致 |
| F-08 | 周报导出 | 进入周报页，点击复制摘要 | 剪贴板写入摘要，内容包含情绪、护理、预警、媒体播放信息 |

## 服务端

| 编号 | 场景 | 步骤 | 预期 |
| --- | --- | --- | --- |
| S-01 | 工作台 | 进入服务工作台 | 待处理工单、重点个案、随访计划从后端同步 |

| S-05 | 联系家属 | 个案详情点击联系家属 | 有手机号时调起系统拨号；无手机号时明确提示 |

## 真机与异常场景

| 编号 | 场景 | 步骤 | 预期 |
| --- | --- | --- | --- |
| D-01 | 老人端大字体 | 真机进入老人端我的页，切换大字体，再回到首页、陪伴、提醒和记录页 | 文字放大后不遮挡按钮和关键内容，触控目标仍可点击 |
| D-02 | 老人端高对比 | 真机切换高对比，再进入首页、心理咨询、媒体播放和求助页 | 背景、文字、按钮对比度提升，状态色仍可区分 |
| D-03 | 录音权限拒绝 | 首次进入 AI 陪伴页时拒绝录音权限，再点击语音输入 | 页面提示开启录音权限，不崩溃；文字输入仍可使用 |
| D-04 | 摄像头权限拒绝 | 进入心理筛查或需要摄像头的页面时拒绝摄像头权限 | 页面提示授权或改用其他方式，不上传空数据 |
| D-05 | 弱网请求 | 在微信开发者工具或真机网络面板设置弱网，刷新老人首页、家属看板和服务工作台 | 出现加载、失败提示或重试入口，不出现长时间空白 |
| D-06 | 离线恢复 | 断网后进入老人首页并尝试刷新，再恢复网络重试 | 离线时显示可理解提示；恢复后能重新拉取数据 |
| D-07 | 语音播放失败 | 屏蔽或删除一条 AI 音频临时 URL 后点击播放 | 页面提示音频不可用，仍显示文本回复 |
| D-08 | 低性能设备滚动 | 在低端 Android 机或开发者工具低性能模式滚动老人首页、媒体列表和咨询师列表 | 滚动可用，无明显布局错位或按钮遮挡 |

## 管理端联动抽查

| 编号 | 场景 | 步骤 | 预期 |
| --- | --- | --- | --- |
| A-01 | 管理端健康检查 | 打开后台管理端，进入系统设置，点击日志管理 | 弹窗展示 Java 后端 `/api/health` 状态 |
| A-02 | 用户管理 | 进入用户管理，新增用户、编辑姓名/电话、删除一位家属、搜索用户、切换风险等级、导出数据 | 列表来自 `/api/users/{familyId}`；新增调用 `/api/users`，编辑和软删除调用 `/api/users/{userId}`，删除后老人端联系人同步移除 |
| A-03 | 内容管理 | 上传一张图片或视频 | 调用 `/api/family/media` 上传，刷新后列表出现新内容 |
| A-04 | 预警处理 | 进入预警中心，筛选未处理并点击立即处理 | 调用后端处理接口，刷新后进入已处理状态 |
