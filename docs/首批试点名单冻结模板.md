# 首批试点名单冻结模板

适用范围：封闭试点启动前，由试点负责人补齐真实姓名或内部编号，并在本文档底部确认冻结。未冻结名单不得进入真实试点。

## 试点家庭

| 家庭编号 | 老人内部编号 | 家属内部编号 | 绑定状态 | 家属联系电话 | 风险备注 | 是否进入试点 |
| --- | --- | --- | --- | --- | --- | --- |
|  |  |  | 未绑定 / 已绑定 |  |  | 否 |

最低要求：

- 10-30 户真实家庭。
- 每户至少 1 位老人和 1 位家属。
- 家属已完成用户协议和隐私政策同意。
- 老人端、家属端均完成首次登录和家庭绑定确认。
